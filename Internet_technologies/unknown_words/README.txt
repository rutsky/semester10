Simple web site for learning unknown words from movies subtitles.
This is the course project for Internet technologies lessons in university.

Copyright (C) 2011  Vladimir Rutsky <altsysrq@gmail.com>

To setup local development environment:
 1. Clone django-nonrel and setup links by running clone_django_nonrel.sh 
    script.
 2. Download latest Google App Engine Python SDK from 
    <http://code.google.com/appengine/downloads.html> and unzip 
    `google_appengine' directory into `server/.google_appengine'.
